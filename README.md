Первым делом склонировать репозиторий:

git clone https://github.com/TemiyLitvinov/Laba4 cd Laba4

Активация venv: 
Windows: venv\Scripts\activate
MacOS/Linux: source ./.venv/bin/activate

Чтобы сделать нового бота, необходим @BotFather в Telegram. Копируем из бота токен и вставляем его в TELEGRAM_TOKEN
Инструкция по работе с ботом:

/start - запускает бота
/help - выводит доступные команды
/game - выдает базовую информацию об игре и её описание
/top - показывает текущие топ 10 игр
/requirements - показывает системные требования, необходимые для игры.
